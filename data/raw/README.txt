Purpose of this folder

This folder stores the original DVF datasets downloaded from the official French open data portal.

The raw data files are not included in this GitHub repository due to their large size.

Expected files:
- valeursfoncieres-2021.txt.zip
- valeursfoncieres-2022.txt.zip
- valeursfoncieres-2023.txt.zip
- valeursfoncieres-2024.txt.zip
- valeursfoncieres-2025.txt.zip

To reproduce this project, download the datasets and place them in this folder.
