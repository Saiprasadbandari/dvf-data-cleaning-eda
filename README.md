# dvf-data-cleaning-eda
# DVF Data Cleaning and Analysis

## Objective

This project uses the French DVF (Demande de Valeurs Foncières) open dataset to perform data cleaning, exploratory data analysis, and feature engineering using Python and Pandas.

## Tasks

- Data Collection
- Data Cleaning
- Missing Value Handling
- Outlier Detection
- Exploratory Data Analysis (1D and 2D)
- Feature Engineering

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Git/GitHub
